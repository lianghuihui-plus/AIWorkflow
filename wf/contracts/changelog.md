# 产物契约：CHANGELOG.md

## 路径

`CHANGELOG.md`

## 职责

归档已解决的人工决策。它不是普通进度日志。

## 条目格式

```markdown
## YYYY-MM-DD

### HH:MM — {决策摘要}（来自 ISSUES.md Q-001）

- **问题：** {原问题文本}
- **决策：** {人工决策}
- **影响：** {原影响文本}
- **处理：** {已更新产物}
```

## 写入规则

- 只有用户明确给出决策后才写入。
- 保留原问题和原影响文本。
- 归档后从 `ISSUES.md` 删除对应问题。
- 同时追加一条 `JOURNAL.md` 记录。
