# 工作空间概览

对话输出当前项目进度快照。不生成文件。

> 工具型 skill。在 workspace 目录下执行。

## 执行流程

### Step 1：读取文件

读 `CONTEXT.md`。不存在 → 报错：**「未找到 CONTEXT.md，当前目录不是 AIWorkFlow 工作空间」**，终止。

读 `ISSUES.md`。不存在 → 待决策数均记为 0。

从 CONTEXT.md 提取：项目名称、当前阶段、下一步、平台。
从 ISSUES.md 提取：各阶段待决策问题数（按 `## 分析阶段` 等标题分组，统计 Q-xxx 条目）。

### Step 2：输出概览

```
# 工作空间概览

项目：XXX | 平台：XXX | 扫描时间：YYYY-MM-DD HH:MM

## 进度

当前阶段：XXX
下一步：XXX

## 待决策

| 阶段 | 待决策数 |
|------|---------|
| 分析阶段 | X |
| 设计阶段 | X |
| 实现阶段 | X |
| 测试阶段 | X |

> 共 X 项待人工决策，详见 ISSUES.md
```

### Step 3：输出建议

简短建议（不超过两句话）：

- 有未决策项：指出哪个阶段待决策最多，建议优先处理
- 无未决策项：按工作流顺序推荐下一个未完成的阶段
